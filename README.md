# SPSB Finance Dashboard
Finance analytics dashboard for SPSB Construction Pvt. Ltd., built as an MBA Business Analytics summer internship project.

## Overview
The app presents FY 2025-26 mock financial data for a construction business, including revenue, expenses, profit, cash flow, project performance, invoices, clients, vendors, loans, assets, analytics, reports, and settings.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- Framer Motion
- Lucide React
